Hangman C++

Hangman game created using C++, game allows player 5 turns to guess a country name; An Array is used to contain all the countries to be guessed from. user can add or remove as many countries as possible. number of guessses can be modified. The stars show how many letters the word contains, each letter is revealed when guessed correctly. So far it works via the terminal. Will create an interface for the game soon.
